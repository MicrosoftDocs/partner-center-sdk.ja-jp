# <a name="device-deployment"></a>デバイスの展開

**適用対象:**

- パートナー センター
- Microsoft Cloud ドイツのパートナー センター

- [指定された顧客の構成ポリシーを更新する](update-a-configuration-policy-for-the-specified-customer.md)
